# LaTeX Beamer with Pandoc

## Features
This is my personal repo for making presentation slides.
It supports

- Pictures with captions
- Scientific plots
- References

all written in Markdown (some sections in TeX).

## Usage

1. Install pandoc and LaTeX
1. Run `make`
